# Excel VBA KPI Dashboard

An Excel-based KPI dashboard with VBA automation for executive reporting and performance tracking.

## Features
- KPI dashboard layout (executive-ready)
- VBA automation for refresh / calculations
- Designed for finance and performance reporting workflows

## How to Use
1. Download the Excel file
2. Open in Microsoft Excel
3. Enable macros (if prompted)
4. Review/update the input data sheets
5. KPIs and visuals update automatically
